## ⚡ The External Engine

Exilium is a modern, external assistance tool built from the ground up for Counter-Strike 2. It operates entirely externally, reading memory structures securely over the perimeter without injecting code into the game process, ensuring maximum safety and operational discretion.

### 🎯 Intelligent Targeting (Humanized Aimbot)
Advanced ballistic trajectory modeling featuring dynamic bezier-curve smoothing algorithms and distance-based speed deltas. This ensures entirely indistinguishable human-like input emulation with built-in micro-jitter.

### 🔧 Adaptive Recoil Mitigation (RCS)
Sophisticated, fully customizable delta-based vector compensation that perfectly counteracts weapon spray patterns. Seamlessly integrates with innate cursor movements for fluid spray control.

### 👁️ Tactical Visual Overlays (ESP & Radar)
High-performance, low-latency visual intelligence encompassing precise entity skeleton tracking, 2D strategic radar mapping, health bars, weapon ESP, and explosive detonation timers.

### 🎒 Weapon-Class Matrices
Granular, class-specific configurations allowing for meticulous tuning across diverse engagement scenarios. Independent profiles for:
- Pistols
- Heavy Pistols (Deagle/R8)
- SMGs
- Rifles
- Snipers & Auto-Snipers

### 🔭 Advanced Sniper Integration
Specific logic designed for the AWP and SSG, featuring *Auto Shot* (fires when perfectly aligned), *Scope Only* activation, and precisely configurable *Aim Delay* to simulate natural human reaction times.
