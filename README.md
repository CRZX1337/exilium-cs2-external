## 🌌 The Frontend Architecture (V10)

This repository hosts the official landing page and deployment center for the Exilium CS2 External client. The website has been entirely rebuilt utilizing modern web technologies to reflect the premium nature of the software it distributes.

### Aesthetic Design
- **True Black Matrix Paradigm**: Operating on an absolute black `#000000` canvas to maximize the contrast of vibrant neon green (`#39ff14`) accents.
- **WebGL Interactive Grid**: A pure CSS/JS HTML5 canvas background featuring a dynamic, mouse-reactive particle grid that adds fluid depth.
- **Glassmorphism UI**: Heavy utilization of frosted, translucent dark panels (`bg-black/40` with `backdrop-blur-xl`) suspended above ambient, rotating blurred orbs.
- **Chromical Glitch Overdrive**: Custom `@keyframes` driving subtle chromatic aberration and vibration states on primary interaction buttons.
- **Micro-Audio Reactive Engine**: Subtle, synthesized high-frequency auditory feedback on user hover states engineered via the Web Audio API.

### Core Technologies
- **React.js & Vite**: Fast, modular component architecture with instant HMR.
- **TailwindCSS**: Utilitarian styling for complex gradients, text-glows, and immediate mobile-responsive layout breaking.
- **Framer Motion**: State-of-the-art physics-based animations driving card tilts, magnetic cursor attraction, and smooth text reveals.
- **GitHub Actions (CD/CI)**: Automated deployment pipeline straight from the `main` branch to the `gh-pages` environment.

## ⚡ The External Engine

Exilium is built from the ground up to provide a premium advantage without compromising account integrity. Operating entirely externally, it reads memory structures securely over the perimeter.

### 🎯 Intelligent Targeting
Advanced ballistic trajectory modeling featuring dynamic bezier-curve smoothing algorithms and distance-based speed deltas, ensuring entirely indistinguishable human-like input emulation.

### 🔧 Adaptive Recoil Mitigation
Sophisticated delta-based vector compensation that seamlessly integrates with innate cursor movements for fluid spray control.

### 👁️ Tactical Overlays
High-performance, low-latency visual intelligence encompassing precise entity skeleton tracking, strategic radar mapping, and explosive detonation timers.

### 🎒 Weapon-Class Matrices
Granular, class-specific configurations allowing for meticulous tuning across diverse engagement scenarios (Pistols, SMGs, Rifles, Snipers).
